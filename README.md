# OperaWiki

这里起先是一个关于Opera公司及其产品的知识网站，由于其创始人不过居士不满维基媒体的协议更改，脱离维基百科独立而成，辗转迁移至此。但之后也陆续发表了一些相干不相干的其它条目，逐渐形成一个以Opera为中心的IT综合百科网站。本站全部内容依据《[学术著作权责声明](LICENSE)》发表，这个声明与GFDL和CC-by-sa都是双向不兼容，在没有原始版权所有者授权的情况下，是不会把维基百科的内容直接复制而来。更具体的说明，请参见《[关于版权](operawiki-copyright.md)》。
