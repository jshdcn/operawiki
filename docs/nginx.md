# nginx

nginx，按照其作者Igor Sysoev的意见，读作“engine-x”，是一款支持HTTP、反向代理和[IMAP](imap.md)等邮件协议的服务器软件。

2002至2004年间为当时的俄国第二大网站Rambler.ru所开发，主要开发者也是俄国人。由于其采用了类BSD许可协议，包括新浪、网易、[UC](ucweb.md)、[FastMail](fastmail.md)在内的国际大站都对其进行了裁剪利用。

与主要竞争对手[Apache](apache.md)、IIS相比，nginx的量级较轻，模块化较强，较多的工作需要通过配置进行。

虽然大部分nginx用户使用的是[Linux](linux.md)或BSD平台，但是它也发布了[Windows](windows.md)、Solaris的版本。
