# 360浏览器

360浏览器由奇虎公司出品，凤凰工作室代工，其[Windows](windows.md)版本市场占有率高居中国第二。

与其原型[世界之窗](theworld.md)一样，360浏览器在Windows下也分为两个版本，单一[Trident](trident.md)内核的由于绑定了自身或外部的诸多安全服务，称为“360安全浏览器”。IE和[Chromium](chromium.md)双核的则称为“360极速浏览器”。不同的是，360浏览器的书签同步等自带服务绑定的是360帐号。

2011年后，凤凰工作室逐渐将主要精力转向360浏览器，并相继推出了[iOS](ios.md)和[Android](android.md)版。iOS平台另有一个支援[Firefox Sync](firefox-sync.md)的“360 Web Browser”，与之不是一个东西。
