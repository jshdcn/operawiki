# Firefox Sync

Firefox Sync最初称为Weave，是[Mozilla基金会](mozilla.md)为应对Google撤除Browser Sync支持所发起的一项开源专案。它提供了一个同步框架，可以同步浏览器的书签、密码以及任何开发者定义的数据类型，著名的笔记扩展QuickFox Notes就使用Firefox Sync来同步笔记。Weave和[Foxmarks](xmarks.md)、Google Toolbar是Google最早推荐的三款Browser Sync替代应用，但现在只剩下Chrome浏览器。

起初以实验性扩展形式发行，在项目的早期阶段，几乎任两个版本都互不兼容。后来随着协议的不断成熟，终于更名为Firefox Sync并集成进了[Firefox](firefox.md) 4和[SeaMonkey](seamonkey.md) 2.1。

除此之外，Mozilla还专门发布了一个浏览器Firefox Home以将Firefox Sync延伸到[iOS](ios.md)平台，因为苹果的相关限制，这款浏览器是[WebKit](webkit.md)内核的。

由于其开源属性，任何人都可以架设Firefox Sync服务器，也可以编写Firefox Sync客户端。曾经支持Firefox Sync的第三方浏览器主要有iOS上的360 Web Browser和[iCab Mobile](icab.md)。放弃支持的原因与[Epiphany](epiphany.md)类似，即不愿配合Mozilla试图推广OAuth认证而收紧的管控。基于Firefox换壳的浏览器大多兼容Firefox Sync，但也有如[Midori](midori.md)因隐私和性能考虑而移除支持。
