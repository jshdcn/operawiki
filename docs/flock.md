# Flock

Flock是基于[Firefox](firefox.md)的Web 2.0浏览器，从3.0版本起转为基于[Chromium](chromium.md)，由同名企业开发。其主要竞争对手是同样基于Chromium的RockMelt。

除支持Firefox的所有原始功能外，Flock还提供：

* 与图片网站的整合，可以便捷上传图片和浏览上传的图片；
* 与视频网站的整合，可以便捷上传视频和浏览上传的视频；
* 与书签网站的整合，可以同步浏览器书签，只支持Delicious；
* 与交友网站的整合，可以即时获取好友最新状态；
* 与博客网站的整合，可以方便地在本地写博客并发表；
* 与微博网站的整合，可以随时书写感想心得；
* 与邮箱网站的整合，可以定时获取邮件信息，但仍需通过网页收取；
* 与聊天网站的整合，可以在浏览器端进行即时通讯，只支持Facebook。

Flock属于独立公司运作，但与AOL的关系颇不寻常。AOL终止网景浏览器支持时，就把Flock与Firefox并列作为推荐代替品，甚至对于网景早期版本只推荐Flock。其时较公认Flock是[Gecko](gecko.md)核心第二大浏览器，而在部分调查中网景浏览器使用率却稳步攀升，以致有人怀疑Flock暗自将自己标称为Netscape。

Flock可以完全兼容Firefox的扩展，也提出了自己规模很小的扩展机制。但是由于功能增加导致术语增多，Flock的i18n计划远远落后于Firefox，最新版往往只有英文。

2011年Flock公司被游戏公司Zynga收购，旋即放弃了对浏览器的支持，并推荐用户改用Chrome或Firefox。
