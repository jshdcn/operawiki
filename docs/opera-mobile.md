# Opera Mobile

Opera Mobile是由[Opera软件公司](asa.md)开发的，用于在智能手机和个人数码助理上浏览网页的网页浏览器。

Opera Mobile的开发理念与桌面版基本相同，本地直连解析，全网页显示。唯缩减了客户端、服务器等附加功能。

Opera Mobile支持多种[操作系统](os.md)，包括[Android](android.md)、MeeGo、[Windows](windows.md)、MTK、BREW，以及不再提供更新版本的Psion、Windows Mobile、S60、UIQ、Maemo等。支持的语言多达40种。

## 预先安装

不同于同门的Opera浏览器和Opera Mini，Opera Mobile最主要也是最早的发行方式是预先安装，并以此向硬件厂商收取一笔一次性费用。

2000年，为Psion操作系统定制的浏览器被称为Opera Mobile。

2001年，索尼爱立信公司推出UIQ操作系统，选中Opera作为合作伙伴。直至UIQ停产，Opera Mobile始终是其预置浏览器，使得UIQ成为当时网页浏览性能最佳的操作系统之一。

2003年，Opera Mobile被移植到Windows Mobile平台，随即开始了预装活动。由于性能远胜自带的IE Mobile，且价格比用户单独购买便宜很多，厂商对此趋之若鹜。预装Opera Mobile的知名WinMobile机型有：

* HTC Touch Diamond
* HTC Touch Diamond2
* HTC Touch Pro
* HTC Touch Pro2
* HTC Touch HD
* HTC HD Mini
* HTC HD2
* Samsung i900 Omnia
* Samsung i8000 Omnia II
* Sony Ericsson XPERIA X1

2007年，诺基亚与Opera达成合作，在其N90等S60手机上预装Opera Moible，取得不俗销量。

2008年，魅族斥资40万欧元选用Opera作为其第一款手机M8及其操作系统Mymobile的预置浏览器，虽然M8销量出色，这一版本的Opera Mobile却遭致很大批评。

2009年，创新公司宣布了Plaszma OS计划，并采用Opera Mobile作为该操作系统的预置浏览器。

同年12月，Opera宣布自己的BREW和Android版本已准备就绪，坐等厂家上门。唯应者寥寥。

2010年，Opera与联发科达成合作，使Opera Mobile成为下一代MTK平台可选的预置浏览器之一。

## 付费下载

2003年4月27日，Opera Mobile被移植到诺基亚S60平台并提供公开下载。

2003年10月27日，发布6.10，更新为多语言版本。

2005年7月13日，发布8.0，首次支援DHTML。

2005年11月14日，发布8.5。

2006年4月5日，发布8.6，首度支援[SVG](svg.md)。

2006年8月29日，发布8.65，这是Opera Mobile历史上最后一个直接面向终端用户收费的版本。

虽然9.x版本已经以beta的永久方式实质免费，但8.65版本的销售直到10.x版本面世后才正式终止。

付费版本的Opera Mobile有30天的免费试用期，正式使用需支付每台24美元。

## 免费下载

2007年2月，发布9.0 beta。这是Opera Mobile第一个免费下载的版本。

2010年3月16日，发布10.0。这是Opera Mobile第一个免费下载的正式版。

2010年5月11日，第一个Maemo版本问世。

2010年11月9日，第一个Android下载版本问世。

2011年3月22日，发布11.0。针对触控式屏幕和平板电脑作出优化；第一个MeeGo和Windows 7版本问世。

2012年2月27日，发布12.0。这是最后一个以Opera Mobile命名的大版本，此后便不再区分桌面和移动端，而统称Opera浏览器。
