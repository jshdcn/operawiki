# SVG

SVG全称可缩放矢量图形（Scalable Vector Graphics），是由W3C定义的网络图形的[XML](xml.md)表示，类似于表示数学公式的MathML。

在SVG发展的早期阶段，不同机构曾开发了不同的检视器，总称SVG Viewer。这些检视器一般都提供NPAPI和[ActiveX](activex.md)两个版本的浏览器插件。

比较著名的SVG Viewer厂家有Adobe、IBM、Corel和CSIRO。其中Adobe SVG Viewer用户最多，CSIRO SVG Viewer是最早的开源实现，这两个检视器被认为显示效果较好。

但从五大浏览器纷纷内置SVG支持之后，独立的SVG Viewer已无栖身之地。
