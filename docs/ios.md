# iOS

iOS是由苹果公司推出的触控屏幕操作系统，适用于手机（iPhone）、PDA（iPod Touch）和平板电脑（iPad）。iOS的商标权原为思科公司所有，后分享给苹果使用。

在iOS设计之初，曾经有过两套技术路线，一是增强版的iPod、二是简化版的Mac。经过竞争，福斯特领导的第二种方案胜出，因此iOS与Mac OS共享许多重要基础，如Darwin内核、Obj-C开发环境、WebKit软件框架、界面设计（通知栏、Dock）等。

使用Obj-C的本地开发接口并不是一开始就提供给开发者的。iOS 1是一个纯粹的浏览器级操作系统，也是当时销量最大的浏览器级操作系统。开发者只能在网站端为iOS进行扩展，对于硬件的调用受到很大限制。

在开发者和用户的强烈要求下，从第2版起，苹果开放了iOS的本地开发接口，并配套了经审核的App Store机制，以保护系统的安全性。出乎意料地，App Store迅速成为了世界上最大的应用程序商店，为苹果和开发者每年提供数十亿美元的营收。而App Store反过来，又大大帮助了iOS设备，包括iPad的发行。这使得iOS超过Mac OS，与[Android](android.md)、Windows一道雄踞世界三大操作系统之列。

实际上从第4个版本起，iOS才叫iOS，之前的苹果并没有要给其移动操作系统取一个名字，偶尔语焉不详地称之为iPhone OS。

从第13个版本起，iPadOS正式独立，随着iPod Touch停产，此后的iOS特指iPhone上的OS。
