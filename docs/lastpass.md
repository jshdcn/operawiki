# LastPass

LastPass是全球重要的密码同步工具供应商。面向[Opera](browser.md)、IE、[Firefox](firefox.md)、Chrome和Safari提供密码存储及同步服务。其付费账户更可用于包括[Android](android.md)、[iOS](ios.md)、Windows Phone、Windows Mobile、BlackBerry OS、[webOS](webos.md)、Symbian在内的诸多移动平台。

2010年11月Opera甫一推出[扩展平台](extension.md)即支持，而[Opera Link](link.md)迟至次年6月才支持密码同步。

次月，通过收购成为书签同步工具[Xmarks](xmarks.md)的母公司，但Xmarks至今犹未支持Opera。

2011年3月，LastPass服务器被攻陷，大量用户主密码遭窃。
