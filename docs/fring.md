# fring

fring是来自以色列的复合通讯协议手机聊天软件，提供各种智能手机和Java版本。以[iPhone平台](ios.md)的第一款Skype应用而闻名于世。

但随着Skype在移动领域的扩张双方产生了冲突，2010年7月终止了合作，并互相指责。9月，fring发布了自己的VoIP解决方案fringout，并在之后一直保持了到世界主要国家最便宜的网络电话费率。

尽管其音质在低速网络下十分清晰，支援的平台和协议之广也堪称同类软件之翘楚。但因为移动平台使用环境受限，同时社群的建设也不成功，fring仍主要在VoIP发烧友间流传。

2013年一家名为Genband的公司收购了fring，从此聚焦于企业通讯业务，停止了fring品牌下的个人用户服务。
