# WebGL

WebGL是一个透过DOM提供的OpenGL ES的[API](api.md)，通过HTML5的Canvas显示，可使用JavaScript进行控制。其主要目标是在浏览器中展现3D效果。

WebGL的规格最早由[Mozilla基金会](mozilla.md)提出，后来交由计算机图形学领域最重要的组织Khronos集团管理。

相比起其它的3D解决方案如[VRML](vrml.md)，WebGL的最大优势在于被许多主流浏览器直接支持，无需安装插件。包括[Firefox](firefox.md) 4.0及以上、Safari 5.1及以上、Chrome 8.0及以上和[Opera](browser.md) 12.0及以上。

一些主流站点，如Google Maps、Google Books，纷纷开启了WebGL版面。
