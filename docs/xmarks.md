# Xmarks

Xmarks是一个在线同步浏览器书签和密码的工具，支持[Mozilla Firefox](firefox.md)、Google Chrome、[Windows](windows.md)的IE、[Mac](macos.md)的Safari并间接同步到[iOS](ios.md)。Firefox版和Chrome版还可以使用自己的服务器，通讯采用标准WebDav协议。如果不用自己的服务器，就要在它的服务器上注册一个帐户，它提供一个网页界面，移动浏览器亦可使用。

旧称Foxmarks，是Firefox下著名扩展，后来添加了跨浏览器支持后改为今名。对IE和Safari的支持限制较多。

2010年9月，为寻找买家，Xmarks宣布自己由于经营不善即将关闭。12月，密码同步工具供应商[LastPass](lastpass.md)将其兼并。并立即宣布了一项付费服务，每年支付12美元可使用Xmarks为[Android](android.md)、BlackBerry OS和iOS专供的浏览器以及Dolphin Browser扩展。

尽管LastPass在[Opera](browser.md)推出扩展功能后不久便提供了同步工具，但Xmarks的Opera版却迟迟按兵不动。
