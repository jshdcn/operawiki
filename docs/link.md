# Opera Link

Opera Link是一个书签笔记同步工具，也是My Opera的一种特殊客户端。其功能内置于4.0以降版本的[Opera Mini](opera-mini.md)，9.5以降版本的[Opera浏览器](browser.md)，以及10.0以降版本的[Opera Mobile](opera-mobile.md)，对其它浏览器提供一个网页界面。

Opera Link帐户系统与My Opera、Opera Dev和晚近发布的[Opera Unite](unite.md)兼容。早年的Opera Web Mail帐号也可直接登入四者，但现在其帐户系统已被隔离。

2007年11月7日发布的Opera Mini 4是首个支持Opera Link同步的正式版客户端。通过个人注册的帐户，免费在Opera服务器上保存书签、笔记、快速拨号，并可在多种上网终端同步个人帐户内的信息。

2008年5月13日的4.1版本发布后，Opera Link还支持在联动的服务器（指空中服务器、欧美服务器或中国服务器三选其一）同步消息来源（指RSS/Atom）订阅，以及在联动的服务器和相同语言版本的客户端同步搜索。

2008年6月12日，Opera 9.50正式版步其后尘，提供了书签、笔记、快速拨号和个人栏的同步，其中前二项与Opera Mini共用。

2008年10月18日，Opera 9.60正式版引入了搜索的同步，但与Opera Mini的搜索同步各自独立。

2010年3月16日，Opera Mobile 10.00正式版提供书签、快速拨号和搜索同步，与Opera共用。

2010年11月3日，Opera Mini 5.1正式版去除了笔记和消息来源订阅功能，快速拨号开始与Opera和Opera Mobile同步。

2010年12月16日，Opera 11.00正式版引入内容阻止规则同步。由于这一版本擅自改变了快速拨号的排列方式，使得之后数月内所有同步客户端的快速拨号顺序颠三倒四，其影响至今犹未消除。

2011年3月22日，Opera Mini 6.0正式版恢复了消息来源订阅和同步功能，并能通过Opera Link单向导入Opera和Opera Mobile的搜索引擎列表。

2011年6月28日，Opera 11.50正式版启动了密码同步。

迄今为止，[Opera软件公司](asa.md)既未开放浏览器同步的接口，也没有开放Opera Link的接口。导致一方面Opera浏览器不能够采用[Firefox](firefox.md)等平台流行的[Xmarks](xmarks.md)、Delicious等书签服务实行自动同步，即使同时拥有Opera Link帐号也不行；另一方面没有第三方程序能够共享Opera Link同步内容。K-Meleon等浏览器虽然可以存取Opera桌面书签，但如没有开启Opera浏览器，依然无法享用Opera Link的便捷服务。包括Opera Mobile在内的不受支持浏览器只能通过登录link.opera.com来访问和修改书签、笔记，在屏蔽My Opera的国家连这样做也煞费周折，对于快速拨号、搜索、个人栏、消息聚合和输入历史更是束手无策。

在10.00的部分测试版本中，opera:config开始将Opera Sync Data Provider设为可选项，但尚未见报称将其改为myopera以外的值会有任何效果。

在安装新版本时Opera会预置含有广告性质的书签和快速拨号，如未先将其清空即启用Opera Link，这类广告书签会自动并入同步列表并删除用户建立的同名书签。
