# 关于外链

**关于链入**

本站的内容根据HDL发表，如果您需要在自己的作品中解释某个字词，又认为本站的词条值得采信，欢迎把链接指向本站，我们不作任何限制。

如果同时需要摘抄部分的词条内容，请注意遵守HDL的相关规范。

**关于链出**

本站无意于成为大而全的百科全书，收录的词条多是因为网络上没有以合理的授权、合理的地址发布的合理的内容。如果网络上的内容符合我们的质量要求，我们愿意将相关链接指向这些优质内容，而不是重造车轮。

我们的质量要求包括如下条件：

1. 内容授权允许接受这样的外链；

2. 以维基方式编写并不对作者资格进行限制，质量在类似网页中出类拔萃者不受此限；

3. 以解释词条为主要目的；

4. 内容的主体部分由简体中文写成；

5. 具备稳定可靠的URL，在不特定的IPv4和IPv6网络下均能正常访问。

截至当前，我们接受的链出站点包括：

* userbase.kde.org

* darksair.org/wiki/

* zh.wikipedia.org
