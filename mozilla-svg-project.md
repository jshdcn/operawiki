# Mozilla SVG Project

Mozilla SVG Project是Mozilla为Gecko引擎增加SVG支援的计划，从Gecko 1.8.0开始试验，到Gecko 1.9.0基本竣工。

最早的SVG查看器都是以NPAPI插件形式存在，例如Adobe Viewer和Corel Viewer。Mozilla的目标就是替代这些插件，让Firefox和SeaMonkey原生支持SVG。第一个正式版的实现是搭载Gecko 1.8.0的Firefox 1.5，而在搭载Gecko 1.9.0的Firefox 3.0中，由于引入了cairo图形引擎，使得SVG的特性实现大大加强，其代价是牺牲了对Windows 98和Me、Mac OS 10.2和10.3的支持。
