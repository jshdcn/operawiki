# Chrome OS

Chrome OS为Google公司私有，是目前商业化程度最高，也是现存最重要的浏览器级[操作系统](os.md)。与Chrome一样，它也是来源于一个完全自由的Chromium OS。

与其它浏览器级操作系统类似，HTML、CSS和JavaScript乃其三大主力编程语言，但Chrome也在试图支持一部分用C或C++编写的[Native Client](nacl.md)。相比较其先驱者[webOS](webos.md)，Chrome OS更加完全地依赖网络，整个系统就是由若干驱动程序和一个Chrome浏览器所构成。

在Chrome OS项目宣布之前，曾出现过一些概念性的基于[Firefox](firefox.md)的操作系统，如Good OS公司提出的Cloud OS。但是由于推广和服务的问题，没有产生像Chrome OS般的影响。

预装Chrome OS的笔记本电脑称为Chromebook，三星出品的首台Chromebook已于2011年6月正式商用。

对于想要安装在Chromebook上的第三方浏览器来说，有两种潜在的可选路径，一是2016年开始提供的[Android](android.md)运行时，通称ARC，二是2018年开始支持的[Linux](linux.md)容器，又叫Crostini。2019年后此二项已是出厂Chromebook的标配，惟后者仍需要通过开发者模式打开。
