# Opera Widget

Widget是[Opera](asa.md)所设计的一种本地应用程序，基于[WAC](wac.md)平台建立。Widget的内容通常和浏览器本身没有太大关系，也不能共用数据。

在桌面端，[Opera](browser.md) 9.0版本引入了Widget概念，起初是类似于扩展的形式，通过Opera下载、安装和管理，并随浏览器启动。而从10.5开始改为完全独立的本地应用程序，浏览器的主要作用是提供一个下载和安装的界面，管理也可以通过操作系统进行。

在移动端，Opera Widget目前还处于实验阶段。可以运行于Windows Mobile、Symbian和[Android](android.md)平台，由名为Opera Widgets的专门程序进行下载和管理。

不同于扩展平台，Opera官方帐号在Widget平台十分活跃，时常会借此发布一些试验性质的工具软件。例如Calendar和OOrganizer就是其在日历领域的两次尝试，实际起到的作用是与[Mozilla](mozilla.md)的Lightning计划相抗衡。
