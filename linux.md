# Linux

Linux是由芬兰程序员Linus Tolvards创立于1991年的一款操作系统内核。受开源运动的影响，Linus决定将自己的作品用GPLv2许可证发布，并随即为GNU接纳和完善形成完整的工具链GNU/Linux，但2007年GNU决定升级到传染性更强的GPLv3时，Linus选择不再跟随。这一决定同时得到了开源和闭源Linux社团的支持，使其得以继续成长为在服务器、移动端、嵌入式设备中统治性的内核，因其巨大的影响力，常常就被简称为内核。

Linux也仅仅是内核，如果想要搭建整套可用的操作系统，有两类主流的方案，一种是直接利用GNU所提供的脚手架，例如Debian等一系列发行版所做的那样，如果整个系统都由自由软件构成，这也不是什么问题，另一种是完全重写GNU框架，用Apache或更开放的协议代替，从而隔离在上层运行的专有软件，如Google在Android所做的那样，这种方式所获的的操作系统，通常就不再被称为Linux发行版。
