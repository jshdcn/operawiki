# 傲游浏览器

傲游浏览器是由陈明杰领导的MySoft的一个浏览器集群，前后使用两个产品名称，六个语言版本拥有各自独立的主站。在Windows平台上，它包含了七个共用部分核心代码的不同浏览器，其中三个同时开发同时支持。

在开源浏览器MyIE的基础上改写而成，后者的作者是迄今不明身份的网民“畅游”。

傲游0，即MyIE2，始创于2002年，与MyIE的差别不大，MySoft已经停止支持。这个产品是最早开始流行的IE+壳浏览器，开创了中国浏览器产业盛极一时的IE造壳运动，影响了一批追随者：中国的全部浏览器厂商，包括互联网行业领导者新浪、搜狐、腾讯、奇虎都选择IE+壳挂自己品牌的开发模式，并美其名曰“Trident内核”。Trident内核浏览器由于直接调用Windows Internet Explorer渲染网页，无需在浏览器内部自带所需排版引擎，故安装包十分微小。

傲游1，始创于2004年，在MyIE2的基础上添加了Gecko内核，但需要单独安装Mozilla [ActiveX](activex.md) Control，因为两个内核都不是自己的，问题较多，至2010年傲游3发布时仍在开发和支持。

傲游2，始创于2006年，取消了对Gecko内核的支持，但又不同于MyIE2，主要基于IE开发，标志特性是“假死”，用户较多，至2012年傲游4发布时仍在开发和支持。

傲游3，2009年提出，2010年进入正式版，这个版本内置的是WebKit内核，并仍旧可调用IE，是亚洲历史上首个自带内核的主流浏览器。

之所以称为主流浏览器，一方面是在欧盟的5+7浏览器选择界面中傲游榜上有名，另一方面是因为傲游在CNET Webware100大奖评选中两次入围浏览类TOP10，2008年一同当选的是Internet Explorer 7、[Mozilla Firefox](firefox.md)、Opera、Safari，2009年除了IE升级外，又多出了Flock和Google Chrome，另外三个不是浏览器的获奖者是Xmarks、Diigo和iGoogle。

随傲游3发布的，是Android平台的手机浏览器，称为傲游MM。当然Android版本就没有Trident内核了。如果有傲游帐号的话，这一版本已经具备了上传下载书签的功能，相比起同期还不稳定的Weave，进度不算太落后。

2012年发布的傲游4，趋炎附热又称为傲游云浏览器，最大的特性是可以像Chrome Sync、Opera Link一般自动同步书签，并且加入了当时很火的云下载功能。

随傲游4同时发布的，是Mac OS X和iOS版本。这两个平台上的傲游与其他第三方浏览器有着相似的表现，即App Store版本套壳系统浏览器，独立版本基于Chromium。

2014年前后，傲游的影响力达到了顶点，标志性的事件是他们推出了Linux和Windows Mobile版，结果维持了不到一年就停止更新了。

2016年发布的傲游5，支持自动同步密码。其商店版本是Mac App Store上迄今能下载到的最新版傲游。由于系统设计上不支持跨大版本升级，所以现在也不知道如何是好。

2020年，在连IE的继承者Edge都放弃EdgeHTML的情况下，双内核已无任何技术可行性和商业价值，傲游6终于妥协，放弃了自行维护11年的内核基线，加入Chromium阵营，当然，也包括了Chrome Web Store生态。这成了傲游历史上首个支持跨大版本升级的版本，无需重新安装就可升级到2023年发布的傲游7。
